# Customer Churn Prediction Report

## Objective
To predict customer churn using demographic and service data.

## Dataset
Telecom customer dataset containing service, billing, and demographic features.

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Confusion Matrix
- Precision, Recall, F1-score
- ROC-AUC

## Results
Random Forest achieved better ROC-AUC compared to Logistic Regression.

## Business Impact
The model helps identify high-risk customers to enable proactive retention strategies.
